Get the Client

Install the Logographic Chat client on your system and start chatting from the terminal.

Install via Homebrew:

brew tap ike5/logographic-chat brew install logographic-chat

Usage:

logographic-chat login # Authenticate logographic-chat # Start chatting logographic-chat logout # Clear credentials

Account & Authentication

Web-based Account Management

All account operations are handled through the web interface. This ensures security and provides a consistent experience across devices.

  • Account Creation: Sign up via the web interface with email/password or OAuth providers
  • Password Changes: Update your password through the account settings
  • Account Deletion: Delete your account permanently via the web interface
  • Device Authorization: Use device codes to authorize terminal clients

Terminal Client Authentication

The terminal client uses a secure device authorization flow:

  • Run logographic-chat login to get a device code
  • Enter the code on the web authorization page
  • Your terminal automatically receives authentication tokens
  • Credentials are stored securely in ~/.config/logographic-chat/
  • Run logographic-chat logout to remove stored credentials
